# Pomodoro-Timer
a light-weight desktop pomodoro timer app which makes you more productive

<img src="https://i.imgur.com/NrR243C.png" width="500">

<h3>Features:</h3>
<li> A cute, simple and light-weight desktop pomodoro-timer app which makes you more concentrated and productive. </li>
<li> The built-in work time and rest time setting allows you to start your task without hesitation and with only one click. </li>

<h3>Tools & Techniques:</h3>
<li> TKinter </li>

<h3>How to Run the App(on MacOS):</h3>
<li> Clone the entire repo </li>
<li> Open your Terminal: </li>

```sh
$ git clone https://github.com/midnamic912/Pomodoro-Timer.git
$ cd <path of the cloned repo>
$ python main.py
```
